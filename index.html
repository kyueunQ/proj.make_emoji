Project Intention

  • What?

    이모티콘 플랫폼 생성

    누구나 쉽게 제작하고 제안할 수 있는 이모티콘 제작 플랫폼

  • Why?

    이모티콘 수요 증가,
    시장의 빠른 성장

    • 5500종의 카카오 이모티콘,
      10억 매출

    • 매 달 약 2000건 이상의 제안서
    • 서비스보기

  • How?

    개발 툴 및 API

    • API : Pixlr API
    • Framework : Ruby on rails
    • UI : Bootstrap4

  • What we expect?

    손 쉽게 이모티콘 작가되기

    • No expertise, No photoshop!!
    • Easy make, easy suggestion
    • Everybody can be a creator!
